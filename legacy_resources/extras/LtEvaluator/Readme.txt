*****************
*    English    *
*****************

When you launch the program, if you get the following error message :
The language DLL 'VB?FR.DLL' could not be found,

Then move the included file "Vb6fr.dll" into your
C:\Windows directory and relaunch the program.

In certain cases you will have to put this file into
C:\Windows\System directory 

In other cases it will be into
C:\Windows\System32 directory.
---------------------------------------------
If that doesn't work at all,
you will have to download the file Msvbm60.dll
and put it into C:\Windows directory
or into C:\Windows\System directory
or into C:\Windows\System32 directory

Quand vous lancez le programme, si vous obtenez le message suivant :
The language DLL 'VB?FR.DLL' could not be found,

Déplacez le fichier inclus "Vb6fr.dll" dans votre répertoire
C:\Windows et relancez le programme.

Dans certains cas, vous devrez placer ce fichier dans le répertoire
C:\Windows\System

dans d'autres cas ce sera dans le répertoire
C:\Windows\System32
